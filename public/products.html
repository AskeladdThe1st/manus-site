Overview of available agents

Interview Agents

Future Specialized Agents

A growing catalog: onboarding coaches, QA reviewers, operations copilots. Built on the same framework for quick, governable launches.

When to use which agent

  • Tutor Agents: when teams need repeatable learning with progress tracking and scoped drills.
  • Interview Agents: when decisions depend on consistent questioning, scoring, and auditability.
  • Future Specialized Agents: when a focused role is needed for onboarding, QA, or operations support.

Each agent is defined by a role, a boundary, and an evaluation loop. New roles inherit the same guardrails.
